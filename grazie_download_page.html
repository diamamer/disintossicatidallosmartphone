Come installare e usare l’app

  1. Scarica il file ZIP cliccando sul pulsante sopra.
  2. Estrai il contenuto del file ZIP sul tuo smartphone o PC.
  3. Apri il file app.html con un browser moderno (Chrome, Firefox, Safari).
  4. Installa la PWA (se il browser lo supporta), solitamente con “Aggiungi a schermata Home” dal menu del browser.
  5. Goditi l’app senza distrazioni, attiva le notifiche e ritrova il tuo equilibrio mentale!
